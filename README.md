# PolyWrapper

**Professional decal placement and trim sheet workflows for 3ds Max.**

PolyWrapper gives 3D artists a complete toolkit for applying trim sheets, placing mesh decals, and generating strip details — all from a single, themeable interface inside 3ds Max.

![PolyWrapper — Trim Sheet tab with multi-set Quick Sets](docs/Images/TrimSheets.png)

## Supported Versions

- 3ds Max 2024
- 3ds Max 2025
- 3ds Max 2026

## Quick Install

1. Extract the `PolyWrapper_<version>_Max<YYYY>` folder that matches your Max version to a stable location.
2. Open 3ds Max and drag `PolyWrapper_MacroInstaller.ms` into the viewport.
3. The tool launches automatically on first install.

See the [full installation guide](docs/installation.md) for toolbar setup, updates, and troubleshooting.

## Features

| Tool | Description |
|------|-------------|
| [Trim Sheet](docs/trim-sheet.md) | Apply trim textures to faces with manual projection or automatic hotspot placement |
| [Hotspot](docs/hotspot.md) | Auto-assign trims to geometry based on face size and texel density |
| [Mesh Decal](docs/mesh-decal.md) | Interactive point-and-click decal placement with real-time surface conforming |
| [Strip Decal](docs/strip-decal.md) | Generate continuous trim and edge detail meshes along selected edges |
| [Set Creator](docs/set-creator.md) | Build and manage your own decal and trim sheet asset libraries |
| [Quick Sets](docs/quick-sets.md) | Preset slots for fast set switching, with multi-set side-by-side viewing |
| [UV Adjustments](docs/uv-adjustments.md) | Flip, rotate, nudge, scale, and grow/shrink UVs with smooth animations |
| [Material Matching](docs/material-matching.md) | Automatic scene material matching using token-based scoring |
| [Options & Theming](docs/options-and-theming.md) | Color presets, per-tab customization, animated backgrounds, and more |
